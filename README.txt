# Simple Calculator (Python)

## Description
A basic command-line calculator built using Python. It supports addition, subtraction, multiplication, and division.

## Features
- Easy menu-based interface
- Error handling for division by zero
- Simple and clean code structure

## How to Run
1. Install Python
2. Clone this repository
3. Run:
   python calculator.py
